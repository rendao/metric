
## About Metric

> One open source psychological testing and assessment (Psychometrics) system.

### In development...


### Install
```
git clone https://github.com/rendao/metric.git
```
```
cd metric
```
```
composer update
```

config mysql information in `.env`

```
php artisan migrate
```
```
php artisan serve
```
